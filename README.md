# Genius

Um projeto pessoal de um jogo de memória interativo desenvolvido com HTML, CSS e JavaScript.

## Sobre o Projeto
Genius é um jogo clássico de memória onde o computador gera uma sequência aleatória de cores e as exibe. O jogador deve reproduzir a sequência correta. A cada acerto, uma nova cor é adicionada à sequência, aumentando gradualmente a dificuldade. O jogo termina quando o jogador erra a sequência.

Este projeto foi desenvolvido exclusivamente para **explorar e praticar técnicas de desenvolvimento web**, incluindo manipulação do DOM, eventos de clique, animações CSS e lógica de jogo em JavaScript. 

## Como Rodar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Clique no botão **Start** para iniciar o jogo
4. Observe a sequência de cores e reproduza-a clicando nos botões
5. Acompanhe sua pontuação conforme avança nas rodadas

## Estrutura do Projeto

- `index.html` - Estrutura da página
- `Estilo/Style.css` - Estilos e design
- `js/script.js` - Lógica do jogo
